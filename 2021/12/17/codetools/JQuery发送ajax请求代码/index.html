简单的模板

类型参考：

dataType 作用
xml 返回 XML 文档
html 返回纯文本 HTML 信息；包含的 script 标签会在插入 dom 时执行
script 返回纯文本 JavaScript 代码。不会自动缓存结果。除非设置了 “cache” 参数。
json 返回 JSON 数据
jsonp JSONP 格式。使用 JSONP 形式调用函数时,
text 返回纯文本字符串
content-type 作用
text/html HTML格式
text/plain 纯文本格式
text/xml XML格式
image/gif gif图片格式
image/jpeg jpg图片格式
image/png png图片格式
application/xhtml+xml XHTML格式
application/xml XML数据格式
application/atom+xml Atom XML聚合格式
application/json JSON数据格式
application/pdf pdf格式
application/msword Word文档格式
application/octet-stream 二进制流数据（如常见的文件下载）
application/x-www-form-urlencoded
中默认的encType，form表单数据被编码为key/value格式发送到服务器（表单默认的提交数据的格式）
multipart/form-data 需要在表单中进行文件上传时，就需要使用该格式
除此之外，我们可以在后面设置编码类型;charset=UTF-8

理解各种类型的请求

Get请求

URL路径传参

请求路径：http://localhost:8080/api/test?id=100
后端接收:如果存在多个参数，我们可以在@RequestParem中指定参数的名字来指定获取。

URL路径参数

请求路径：http://localhost:8080/api/test/100/gu
后端接收：主要适合Restful风格的请求

POST请求

前端通过Body传值，介绍使用json进行传送数据。
在postman中使用raw，选择json，进行请求模拟。
后端接收：可以使用@RequestBody或者@Requestparam

Tips：application/x-www-form-urlencoded格式上传的数据，后台只能使用@RequestParam来获取参数
