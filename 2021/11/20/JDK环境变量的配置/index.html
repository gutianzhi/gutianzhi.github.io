JDK环境变量的配置

什么是JDK

JDK是 Java 语言的软件开发工具包，主要用于移动设备、嵌入式设备上的java应用程序。JDK是整个java开发的核心，它包含了JAVA的运行环境（JVM+Java系统类库）和JAVA工具。

JDK的安装

大致步骤：

  • 官网下载地址：https://www.oracle.com/java/technologies/downloads/
    根据
  • 根据自己的系统和机型选择合适的版本进行安装
  • 安装时注意安装的位置，个人不喜欢安装在C盘，记住安转后的路径，之后我们在配置环境变量的时候会用到。

环境变量的配置

基本步骤：

  • 右击此电脑，选择属性
  • 在关于中选择高级系统设置
  • 选择环境变量
  • 在系统变量中新添加一个变量，变量名为JAVA_HOME,变量的值就是JDK安装的位置。
  • 接下来我们在Path环境变量中配置我们配置好的JDK的BIN的执行文件地址，主要的目的就是我们就可以通过相关的指令在命令行中使用。

验证

在cmd中打开命令行，输入java -version。如果出现相关版本的信息就证明配置成功。

补充

什么是JRE

Java运行环境（Java Runtime Environment，简称JRE）是一个软件，由太阳微系统所研发，JRE可以让计算机系统运行Java应用程序（Java Application）。
JRE的内部有一个Java虚拟机（Java Virtual Machine，JVM）以及一些标准的类别函数库（Class Library）.

classpath

当我编译好一个java文件后，运行，JVM如何知道这个java文件对应的class文件在哪呢？classpath的作用就是通知JVM用户类的存放路径
