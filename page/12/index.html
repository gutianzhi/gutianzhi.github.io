mysql无法访问java.sql.SQLException:null,message from server:"Host '***' is not allowed to connected
发表于2021-11-21|更新于2021-11-21|BUG|mysql•bug
问题描述最近在服务器中部署项目的时候，我在服务器中安装了mysql数据库，当我把相关数据库导入到服务器然后启动之后，就出现改问题。开始很郁闷，访问地址确实是填的我服务器的ip地址呀，查询之后才发现，数据库原来设置的root只限制localhost、127.0.0.1地址访问。 解决方案 进入cmd，输如以下命令：1mysql -uroot -p 1select user,host from user 1update user set host='%' where user='root'（%匹配任何地址） 1flush privileges 重启服务1win+R之后输入services.msc找到mysql服务器重新启动即可。
